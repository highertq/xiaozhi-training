# 小智AI机器人 · 培训中心

基于ESP32-S3的开源智能语音交互终端，支持DIY，可扩展性强

## 📖 项目简介

小智AI聊天机器人是一个开源的智能语音交互终端项目，基于ESP32系列芯片开发。本项目完全开源，提供完整的固件源码、DIY教程、配置指南，适合个人学习、教育展示、商业应用。

**核心特点：**
- ✅ 完全开源，可DIY
- ✅ 支持多种硬件形态（ESP32-S3、ESP32-C3、ESP32）
- ✅ 可扩展性强（GPIO、I2C、SPI、摄像头、LCD、传感器）
- ✅ 活跃的社区和丰富的教程
- ✅ 最新版本：v2.0.5

**我们的配置：**
- 主控：立创ESP32-S3开发板
- 音频：I2S麦克风 + I2S扬声器
- 软件：小智AI开源固件 v2.0.5

## 📊 硬件配置

### 立创ESP32-S3开发板

| 项目 | 规格 |
|------|------|
| 主控芯片 | ESP32-S3 (乐鑫) |
| 处理器 | 双核Xtensa LX7，240MHz |
| 内存 | 512KB SRAM + Flash |
| 网络 | WiFi + 蓝牙5.0 |
| 音频 | I2S接口（麦克风+扬声器） |
| 外设 | GPIO、I2C、SPI、UART |

### 音频模块

- **麦克风**：I2S数字麦克风，高灵敏度，抗干扰强
- **扬声器**：I2S数字扬声器，内置功放，音质清晰

### 成本分析

| 项目 | 价格 |
|------|------|
| ESP32-S3开发板 | 50-80元 |
| I2S麦克风模块 | 15-20元 |
| I2S扬声器模块 | 15-20元 |
| 面包板 + 杜邦线 | 20元 |
| 电池模块（可选） | 20-30元 |
| **硬件总计** | **120-200元** |

**运营成本**：
- AI API调用：约0.5-1元/千次对话
- 可选择免费模型（如豆包）

## 💡 核心功能

### 1. 语音交互
- **唤醒词**：可自定义（默认"小智小智"）
- **语音识别**：云端ASR（如阿里云、讯飞）
- **语音合成**：云端TTS（多种音色可选）
- **声音复刻**：自定义声音

### 2. AI对话
- **接入大模型**：通义千问、文心一言、豆包等
- **人设切换**：英语外教、故事大王、诗人等
- **多轮对话**：支持上下文理解
- **联网查询**：天气、新闻、百科等实时信息

### 3. 智能控制
- **GPIO控制**：通过语音控制LED、继电器等
- **定时提醒**：语音设置提醒事项
- **联网查询**：通过MCP接入联网搜索
- **OTA升级**：无线更新固件

### 4. 硬件扩展
- **LCD彩屏**：显示表情、文字、图像
- **摄像头**：视频通话、图像识别
- **传感器**：温湿度、光线、运动检测
- **音响外放**：提升音质效果

## 📚 技术架构

```
硬件层：ESP32-S3 + I2S麦克风 + I2S扬声器 + WiFi/蓝牙
  ↓
固件层：FreeRTOS + 语音处理 + 网络通信 + 大模型调用 + OTA升级
  ↓
服务层：语音识别ASR + 语音合成TTS + 大模型API + MCP联网查询 + 配置服务器
  ↓
应用层：语音唤醒 → 语音识别 → AI对话 → 语音合成
```

## 🔧 配置流程

1. **烧录固件**：使用Web端烧录工具或ESP-IDF编译
2. **连接WiFi**：手机连接设备热点，访问xiaozhi.me配置页面
3. **配置API**：输入通义千问等大模型的API Key
4. **测试对话**：说"小智小智"唤醒，然后提问
